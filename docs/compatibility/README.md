# Compatibility Status

This directory is the public compatibility contract for Tansu.

- Kafka 4.2 is the active compatibility baseline.
- Kafka 4.3 is tracked, but it is not a baseline yet.
- Tansu compatibility claims are made per profile and per storage engine.
- No API is fully supported unless the ledger has proof.

The machine-readable source of truth is [kafka-4.2-ledger.json](kafka-4.2-ledger.json).

## Current Contract

- The ledger documents every Kafka broker request generated by `tansu-sans-io`.
- The ledger also documents every API currently routed by the broker service stack.
- `ApiVersions` advertising is still described by the current code path; Phase 02 will tighten that contract.
